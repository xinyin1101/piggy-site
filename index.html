📖 部署指南
1. 创建 Supabase 项目
打开 supabase.com → 用 GitHub 登录 → New Project
记下 Project Settings → API 里的 URL 和 anon public key

2. 创建数据库表
SQL Editor → 新建查询 → 粘贴下面的 SQL → 运行

3. 创建存储桶
Storage → New Bucket → 名称 photos → Public
Bucket 设置里取消勾选 "Restrict file uploads"

4. 填入密钥
把上面的 URL 和 Key 填到页面顶部，点击保存

5. 部署到 Vercel
把这个文件夹传到 GitHub → 导入 Vercel 即可
📋 建表 SQL（复制到 Supabase SQL Editor 运行）
CREATE TABLE messages (
  id bigint GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  text text NOT NULL,
  created_at timestamptz DEFAULT now()
);
ALTER TABLE messages ENABLE ROW LEVEL SECURITY;
CREATE POLICY "所有人可读" ON messages FOR SELECT USING (true);
CREATE POLICY "所有人可插入" ON messages FOR INSERT WITH CHECK (true);
CREATE POLICY "所有人可删除" ON messages FOR DELETE USING (true);
